# Installation

Install RMUI Big Components

## rmui-bigcomponents installation

### npm

```bash
npm install rmui-bigcomponents
```

### yarn

```bash
yarn add rmui-bigcomponents
```

**Be sure to install all the peer dependencies bellow, there are required for the library to work properly.**

## Peer dependencies installation

-   [react](https://www.npmjs.com/package/react) > 18.x
-   [react-dom](https://www.npmjs.com/package/react-dom) > 18.x
-   [@mui/material](https://www.npmjs.com/package/@mui/material) > 5.x
-   [@emotion/styled](https://www.npmjs.com/package/@emotion/styled) > 11.x
-   [@emotion/react](https://www.npmjs.com/package/@emotion/react) > 11.x
-   [@mui/icons-material](https://www.npmjs.com/package/@mui/icons-material) > 11.x

### npm

```bash
npm install react@18.x react-dom@18.x @mui/material@5.x @emotion/styled@11.x @emotion/react@11.x @mui/icons-material@11.x
```

### yarn

```bash
yarn add react@18.x react-dom@18.x @mui/material@5.x @emotion/styled@11.x @emotion/react@11.x @mui/icons-material@11.x
```

You are now able to use rmui-bigcomponents in your project!
